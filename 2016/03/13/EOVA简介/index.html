EOVA简介

  • Eova 是一个 J2EE 快速开发平台。
  • Eova 适用于对 UI 布局和性能没有严苛要求的企业级 WEB 后台管理系统。
  • 是对Jfinal，easyUI的再次封装

EVOA能做什么

  • 简单表数据的显示，分页，无需代码实现。

  • 自定义二级三级菜单，随意DIY。

  • 已经实现的权限管理模块，可自定义用户组，及权限。

  • 登录已实现（目前没有注册功能），可添加用户。

  • DIY查询。
  • 自定义业务（重写业务拦截器）。
  • 自定义按钮（需要html + BS）。
  • 简单配置实现文件上传。

EVOA的优势

  • 减少了代码编写量，简化了开发过程。
  • 易于上手
